大概花费4个小时，不需要掌握后端开发，实现问答小程序Demo。

这不是真正意义上的教程，它是全程开发的实录。

由于我也是第一次使用Bmob开发小程序，意图在于记下每一次的踩坑，因此过程比较冗长，适合快进着看完。

### 技能要点

1、怎么使用Bmob快速实现自己的一个idea
2、基本的CRUD、关联表以及多图上传
3、Bmob集成用户登录
4、Mina框架高频的组件与API的使用
